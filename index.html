List of My Favorite Video Games

Each game is linked with its official website or wiki page.


Multiplayer Games

  • Elden Ring: Nightreign - Multiplayer roguelike game that is a spin-off of the FromSoftware Title, Elden Ring.
  • Valorant - 5v5 class-based tactical shooter.

Singleplayer Games

  • Sekiro: Shadows Die Twice - Singleplayer action adventure from studio FromSoftware.
  • Elden Ring - Singleplayer open-world action RPG from studio FromSoftware.
  • Balatro - Rougelike game based on poker.
